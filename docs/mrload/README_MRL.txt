BigValue MRLoad is a distributed map reduce data loading system over Hadoop and HBase,
specially over Cloudera Distributed Hadoop(CDH).

To get started using BigValue MRLoad, the full documentation for this release can be
found under the docs/mrload directory that accompanies this README_MRL. 

The latest BigValue BulkLoad can be downloaded from an BigValue Mirror [1].
