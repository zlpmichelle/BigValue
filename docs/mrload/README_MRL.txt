BigDataAnalysis MRLoad is a distributed map reduce data loading system over Hadoop and HBase, 
specially over Cloudera Distributed Hadoop(CDH).

To get started using BigDataAnalysis MRLoad, the full documentation for this release can be
found under the docs/mrload directory that accompanies this README_MRL. 

The latest BigDataAnalysis BulkLoad can be downloaded from an BigDataAnalysis Mirror [1].
