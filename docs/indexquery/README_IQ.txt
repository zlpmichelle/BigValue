BigValue IndexQuery is a distributed hbase coprocessor query system over Hadoop and HBase,
specially over Cloudera Distributed Hadoop(CDH).

To get started using BigValue IndexQuery, the full documentation for this release can be
found under the docs/indexquery directory that accompanies this README_IQ.
