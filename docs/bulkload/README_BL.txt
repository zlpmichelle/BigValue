BigValue BulkLoad is a distributed batch data loading system over Hadoop and HBase,
specially over Cloudera Distributed Hadoop(CDH).

To get started using BigValue BulkLoad, the full documentation for this release can be
found under the docs/bulkload directory that accompanies this READMEBL. 

The latest BigValue BulkLoad can be downloaded from an BigValue Mirror [1].
